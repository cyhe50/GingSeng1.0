一種類似於Reigns的卡牌遊戲



遊戲方法：利用對卡牌選擇的結果來影響人物數值，任一數值歸零或超標即結束遊戲，並獲得一種結局




卡牌規劃：四種類別，所有卡牌皆有對應的圖片，部分事件會出現音效
			 
			 
卡牌類別：


	普通卡
	
	主線卡－跟最初選擇的志願有關的事件
	
	餵食卡－與食物有關
	
	鬼牌-強迫進入結局
